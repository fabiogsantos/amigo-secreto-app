Boa noite pessoal... Segue a lista de dependências para as próximas aulas:

Git
Node.js + NPM (Quando você instala o Nodejs, o NPM já vem de boas)
Angular CLI $ npm install -g @angular/cli
Ionic CLI $ npm install -g ionic@latest
Cordova $ npm install -g cordova
Java Development Kit (JDK) versão 7 ou superior
Android Studio
Android SDK
Firebase CLI $ npm install -g firebase-tools
